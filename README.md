# ❤️ Heart Disease Prediction System

A **Machine Learning-powered web application** that predicts the likelihood of heart disease based on patient medical data.  
Built using **Python, Scikit-learn, and Streamlit**, this project provides a simple and interactive medical dashboard for prediction and insights.

---

## 🚀 Live Demo
👉 https://heart-disease-prediction-ss.streamlit.app/

---

## 📌 Project Overview

Heart disease is one of the leading causes of death worldwide. This project uses a **Logistic Regression model** to analyze patient health parameters and predict whether a person is at risk.

The application provides:
- Real-time prediction
- Risk probability score
- Health recommendations
- Clean and user-friendly medical dashboard

---

## 🧠 Machine Learning Model

- **Algorithm Used:** Logistic Regression  
- **Preprocessing:** StandardScaler  
- **Train-Test Split:** 80% / 20%  
- **Evaluation Metrics:** Accuracy, Confusion Matrix  

---

## 📊 Input Features

The model uses the following medical attributes:

- Age  
- Sex (0 = Female, 1 = Male)  
- Chest Pain Type  
- Resting Blood Pressure (BP)  
- Cholesterol  
- Fasting Blood Sugar > 120 mg/dl  
- ECG Results  
- Maximum Heart Rate Achieved  
- Exercise Induced Angina  
- ST Depression (Oldpeak)  
- Slope of ST Segment  
- Number of Major Vessels (Fluoroscopy)  
- Thallium Test Result  

---

## 🎯 Output

- **0 → No Heart Disease (Low Risk)**  
- **1 → Heart Disease Present (High Risk)**  

The app also displays:
- Risk Probability (%)  
- Personalized health suggestions
